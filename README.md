# Overview
This repository contains code, notebooks and supporting documentation for an NYC crime forecasting project. The work begins with a replication of a parent spatiotemporal crime prediction paper and then extends that framework toward richer NYC-specific modeling using cleaned incident-level data.

The project includes:

- data cleaning and merging workflows
- baseline forecasting models
- parent-paper replication work
- point-level extension modeling
- project notes and research references


## Folders
- `Code`: Contains R files for cleaning (no need to do anything with this)
- `notebooks`: Contains colab notebooks that you will need to run the models
- `Data`: Link and Instruction on how to download datasets
